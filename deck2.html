A quick intro to

Responsive Web Design

(aka RWD)

What is RWD?

A design approach that aims for an optimal user experience on all viewing devices with a single code base, employing:

  • Fluid layout
  • Flexible images and resources
  • CSS media queries
  • The Boston Globe

Why use RWD?

  • One website, many devices
  • One user, many devices
  • Single URL
  • Design for devices of the future

Why NOT use RWD?

  • Code complexity
  • Device customization is limited
  • Speed penalties due to unused resources
